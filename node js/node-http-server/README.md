# Node.js HTTP Server

Run `npm i` to install the NPM dependencies, to run the server run `npm run start:dev`.

 ┃ ┗ server.js  // starts the server
 ┣ HOMEWORK.md
 ┣ README.md
 ┣ package-lock.json
 ┗ package.json
